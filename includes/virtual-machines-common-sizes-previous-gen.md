### <a name="standard-a0---a4-using-cli-and-powershell"></a>使用 CLI 和 PowerShell 的標準 A0 - A4

在傳統部署模型中，部分 VM 大小名稱會與 CLI 和 PowerShell 中的稍有不同：

* Standard_A0 是「特小型」 
* Standard_A1 是「小型」
* Standard_A2 是「中型」
* Standard_A3 是「大型」
* Standard_A4 是「特大型」
