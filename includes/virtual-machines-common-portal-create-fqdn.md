
1. 在入口網站中選取您的 VM。 在 [DNS 名稱] 底下，按一下 [設定]。
   
   ![按一下 [DNS 名稱] 底下的 [設定]](./media/virtual-machines-common-portal-create-fqdn/configure.png)

2. 輸入想要的 DNS 名稱，然後選取 [儲存]。
   
   ![輸入您公用 IP 資源的 DNS 名稱標籤](./media/virtual-machines-common-portal-create-fqdn/configure-pane.png)


3. 若要返回 VM 概觀刀鋒視窗，請關閉 [公用 IP 位址] 刀鋒視窗。 確認現在顯示「DNS 名稱」。
