用於新增範例語句的 `response` 陣列會以 `hasError` 屬性來指出每個範例語句成功或失敗。 下列 JSON 回應顯示已成功新增兩個語句。 

```json
    "response": [
        {
            "value": {
                "UtteranceText": "go to seattle today",
                "ExampleId": -5123383
            },
            "hasError": false
        },
        {
            "value": {
                "UtteranceText": "book a flight",
                "ExampleId": -169157
            },
            "hasError": false
        }
    ]
```

以下 JSON 顯示成功訓練要求的結果：

```json
{
    "request": null,
    "response": {
        "statusId": 9,
        "status": "Queued"
    }
}
```
