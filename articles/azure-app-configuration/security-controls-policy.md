# <a name="azure-policy-regulatory-compliance-controls-for-azure-app-configuration"></a>Azure 原則 Azure 應用程式組態的法規合規性控制

[Azure 原則中的法規合規性](../governance/policy/concepts/regulatory-compliance.md)可針對與不同合規性標準相關的 **合規性網域** 和 **安全性控制**，提供 Microsoft 建立和管理的方案定義 (稱為「內建」)。 此頁面會列出 Azure 應用程式組態的 **合規性網域** 和 **安全性控制** 。 您可以針對 **安全性控制** 個別指派內建項目，以協助讓您的 Azure 資源符合特定標準的規範。
