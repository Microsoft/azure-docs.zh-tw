# <a name="knowledge-exploration-service"></a>知識探索服務

歡迎使用 Microsoft 知識探索服務。  知識探索服務 (KES) 提供快速且有效的方法，可將互動式搜尋和精簡搜尋功能新增至應用程式。  透過 KES，您可以從結構化資料建立壓縮的索引，撰寫可解譯自然語言查詢的文法，並提供具有自動完成建議的互動式查詢編寫。

請參閱[使用者入門](GettingStarted.md)中的逐步解說範例，了解如何使用 KES 為為學術出版品領域建立互動式搜尋介面。

![AcademicAutoComplete](AutoComplete.png)

KES 已經從認知服務預覽移至認知服務實驗室。 此專案的新首頁是 [http://labs.cognitive.microsoft.com/en-us/project-knowledge-exploration](http://labs.cognitive.microsoft.com/en-us/project-knowledge-exploration)。
