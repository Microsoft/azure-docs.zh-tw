## <a name="overview"></a>概觀
異常搜尋工具 API 可讓您持續監看資料，並利用機器學習來偵測異常，這種機器學習會藉由自動套用正確的統計模型來適應您獨特的資料。 它讓組織能夠維護資料品質、提供可靠的服務、識別商務事件，以及精簡他們的業務方法，而不需處理手動識別異常資料的棘手問題。
