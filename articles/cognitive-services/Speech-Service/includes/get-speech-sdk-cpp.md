#### <a name="c-nuget-package"></a>C + + NuGet 套件

您可以使用下列命令，從 **封裝管理員** 安裝 c + + 語音 SDK `Install-Package` 。

```powershell
Install-Package Microsoft.CognitiveServices.Speech
```
